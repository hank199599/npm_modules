
說明
-------
改造[24 Point Poker Game Calculator](https://helloacm.com/24/)的PHP代碼，  
轉換為JavaScript的模組形式。

安裝
-------
```bash
npm install 24 points solver
```

使用
-------
```bash
var solver=require('24 points solver')  
solver.calculator(1,2,3,4)

//output will be:
```
